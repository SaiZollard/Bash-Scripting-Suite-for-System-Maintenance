Features

Automated Backups: Safely backs up important directories or files.

System Updates: Performs system updates and removes unnecessary cache/data.

Log Monitoring: Tracks logs for specific error patterns and alerts the user.

Integrated Maintenance Suite: A single script to run all maintenance tasks together.

Cron Integration: Can be scheduled to run automatically at defined intervals.

How to Use

Clone the repository:

git clone https://github.com/SaiZollard/Bash-Scripting-Suite-for-System-Maintenance.git
cd "Bash Scripting Suite for System Maintenance"


Make scripts executable:

chmod +x maintenance_suite/*.sh


Run the maintenance suite:

./maintenance_suite/maintenance_suite.sh


(Optional) Automate with cron:

crontab -e


Add this line to run the suite every day at midnight:

0 0 * * * /path/to/maintenance_suite/maintenance_suite.sh >> /path/to/maintenance_logs/cron_run.log 2>&1


Requirements

Linux or macOS environment (Bash shell)

Root or sudo privileges for system update commands

Basic understanding of shell scripting
